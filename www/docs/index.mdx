Tsynamo simplifies the DynamoDB API so that you don't have to write commands with raw expressions and hassle with the attribute names and values. Moreover, Tsynamo makes sure you use correct types in your DynamoDB expressions, and the queries are nicer to write with autocompletion!

    <td>✅ <b>Type-safe DynamoDB queries</b><br/>
    Tsynamo lets you write type-safe DynamoDB queries. This significantilly reduces maintenence and boosts productivity.</td>
    
    <td>✅ <b>Great DX</b><br/>
    By exposing your DynamoDB schema to the TypeScript compiler, you get autocompletion on table names, secondary indexes, condition expressions and column names etc.</td>
  </tr>
  <tr>
    <td>✅ <b>No magic, just DynamoDB</b><br/>
    Tsynamo is a light abstraction layer over DynamoDB Client making it supereasy to adopt.</td>
    
    <td>✅ <b>Light bundle size</b><br/>
    Tsynamo has zero dependencies making it extremely lightweight and does not pollute your project.</td>
